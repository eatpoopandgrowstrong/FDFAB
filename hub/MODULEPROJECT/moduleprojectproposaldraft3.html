Module Project Proposal 3:

Gas Turbine Engine Mockup

Create a gas turbine engine mockup.

3D Print the engine and shroud

3D print the turbine propeller, attach it to DC brushless motor through bullet adapter

CNC requirement

Option A: 2D CNC a stand for the engine, use it to support.

Option B: 3D CNC foam and create a mold, mold and cast some small part

Electronics

Option A: Create a atmega328p board, interface with IR remote (input) to control DC brushless motor (output)

Option B: Use tiny85 chip and interface with a push button and the brushless motor

Extras (If time permits)

Laser cut a demo box for it, clear acrylic surrounding it

Add LED strips
