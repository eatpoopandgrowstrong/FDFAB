Module Project Proposal

Make an automated standing desk, use a microcontroller to control the height of the desk. Have for example buttons which can be used to set its height. The buttons can be molded and casted.

The table top and supporting legs can be CNC machined from 15mmm plywood. Maybe buy my own tabletop as plywood provided isn't exactly confidence inspiring.

For the microcontroller, make my own Arduino Uno using the Arduino ATMEGA328P, make a custom board with all its pins and mill it from a copper board. Program with the ISP programmer made from embedded programming.

Have a display and maybe use an ultrasonic sensor to measure and display the height of the desk from the ground. User can set certain presets heights to easily recall different heights such as their preferred standing and sitting height, instead of mannually adjusting the table by hand.

Interface the 'arduino uno' with two linear actuators/linear servos. These will allow the table to adjust its height. The linear servos can take 12V, use a laptop power supply to provide the required voltage. Connect the linear servos with a transistor/mosfet. Probably mosfet due to 12V used in this scenario.

Depending on what the IoT topic covers, maybe use some remote or app for a 'smarter' control solution.
